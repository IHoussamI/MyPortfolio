# Houssam Mesk - Portfolio

Modern Angular portfolio with terminal-style UI and French/English support.

- 🖥️ Interactive terminal interface
- 🌐 Multi-language (EN/FR)
- 📱 Fully responsive
- 💾 CV download system

## 🛠 Tech Stack
- Angular 15+ • TypeScript • SCSS
- i18n • RxJS • GitHub Pages
